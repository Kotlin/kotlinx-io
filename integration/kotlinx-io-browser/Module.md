# Module kotlinx-io-browser

The module provides [kotlinx.io.RawSink] and [kotlinx.io.RawSource] implementations for [org.khronos.webgl.ArrayBuffer].

[org.khronos.webgl.ArrayBuffer] could be wrapped into [kotlinx.io.RawSource] using [kotlinx.io.browser.ArrayBufferSource].
Data could be written to [org.khronos.webgl.ArrayBuffer] using [kotlinx.io.browser.ArrayBufferSink].

# Package kotlinx.io.browser

[ArrayBuffer] adapters for `kotlinx-io`.
